My Bookmark project

This is my bookmark project for my AENG110 class. As you can see here I used Adobe Indesign to create this for my class project.

The purpose benind this project was my professor was doing a special delivery for the Eshleman Elemetary schools library having each student make bookmarks for the students to use when they go to the library.

Fisrt I drew out sketches of 5 bookmarks for what design I wanted to make in Adobe indesign, Next I picked the two best design that I thought would best fit for a recreation in Adobe indesign, Finally I finished the bookmarks and got them printed out on white cardstock.

This Project took some time to make but overall I was quite impressed with the finished product of my bookmark for how it came out.
